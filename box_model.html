If you specify a background image, you should also specify a color. The color will not be visible unless the image fails to display for some reason, or unless the imagehas transparent areas in it. The background color is especially important if you use a dark background image and a light foreground color; if the image does not appear, the text still must appear on a dark background to be readable.If you specify a background image, you should also specify a color.
